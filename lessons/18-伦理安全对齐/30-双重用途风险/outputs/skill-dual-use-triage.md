# 双重用途分类技能

## 目标
给定2026年能力声明或事件报告，跨四个领域分类，并识别该声明影响新手相对提升、专家绝对能力还是两者。

## 分类流程

### 1. 领域识别

**四个CBRN领域：**
- 生物（Bio）
- 化学（Chem）
- 网络（Cyber）
- 核（Nuclear）

### 2. 能力声明分析

**新手相对提升：**
- 乘法效应（如2.53倍）
- 相对于现状的帮助程度
- 输入过滤、拒绝可防御

**专家绝对能力：**
- 高上限
- 专家知道问什么以及如何解释
- 需要引出加固、能力遗忘、控制协议

### 3. 2025年阈值跨越

**生物：**
- 获取阶段自动化
- 2.53倍提升
- 接近ASL-3

**化学：**
- 执行差距侵蚀
- 实时湿实验纠正
- 前体采购仍是瓶颈

**网络：**
- 智能体编码工作流
- 80-90%活动自动化
- 4-6个人类干预步骤

**核：**
- 材料获取瓶颈维持
- 信息层提升有限
- 无重大拐点报告

### 4. 安全案例建议

**针对新手相对提升：**
- 输入过滤
- 拒绝策略
- 不确定性表达

**针对专家绝对能力：**
- 引出加固
- 能力遗忘
- 控制协议
- 多层防御

**跨领域综合：**
- 同时评估新手和专家场景
- 考虑执行差距侵蚀
- 定期更新评估
