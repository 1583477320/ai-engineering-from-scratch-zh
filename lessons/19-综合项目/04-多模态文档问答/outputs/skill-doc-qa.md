# 多模态文档问答技能

## 目标
构建视觉优先的多模态文档问答系统，在特定语料库上调优，并在ViDoRe v3上与先OCR再文本基线对比评测。

## 评估标准

| 权重 | 标准 | 测量方法 |
|:---:|------|----------|
| 25 | ViDoRe v3 / M3DocVQA准确率 | 与OCR文本基线和公开排行榜对比 |
| 20 | 证据区域定位 | 引用的区域实际包含答案的比例 |
| 20 | 存储和延迟工程 | DocPruner压缩比、索引p95、答案p95 |
| 20 | 多页推理 | 100个手动标注的多页问题的准确率 |
| 15 | 源文件检查UX | 查看器清晰度、覆盖图精确度、对比工具 |

## 构建检查清单

- [ ] PDF页面渲染（PyMuPDF，180 DPI）
- [ ] ColQwen嵌入（多向量，~2048图像块/页）
- [ ] DocPruner剪枝
- [ ] MaxSim检索
- [ ] VLM答案合成
- [ ] 证据区域提取
- [ ] OCR回退通道（方程式页）
- [ ] ViDoRe v3 / M3DocVQA评测
