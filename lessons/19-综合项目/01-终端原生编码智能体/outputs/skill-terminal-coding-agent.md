# 终端编码智能体技能

## 目标
给定仓库路径和任务描述，在沙箱中运行完整计划-行动-观察-观察循环，返回PR URL和trace包。

## 评估标准

| 权重 | 标准 | 测量方法 |
|:---:|------|----------|
| 25 | SWE-bench Pro pass@1 | 与mini-swe-agent对比30个Python任务 |
| 20 | 架构清晰度 | 计划/行动/观察分离、钩子表面、工具模式 |
| 20 | 安全性 | 沙箱逃逸测试、权限提示、破坏性命令防护 |
| 20 | 可观测性 | trace完整性、每轮token统计 |
| 15 | 开发者体验 | 冷启动<2s、崩溃恢复、Ctrl-C停止 |

## 架构检查清单

- [ ] 计划状态（TodoWrite）每轮完整重写
- [ ] 工具沙箱化，禁止逃逸
- [ ] 八种钩子事件已实现
- [ ] 三层成本控制
- [ ] PreCompact钩子压缩上下文
- [ ] OpenTelemetry trace
- [ ] 沙箱清理（成功或失败）
