# 代码迁移智能体技能

## 目标
给定仓库，执行确定性配方然后智能体循环，产生绿色迁移分支，或将仓库归档到分类桶。

## 评估标准

| 权重 | 标准 | 测量方法 |
|:---:|------|----------|
| 25 | MigrationBench通过率 | 50仓库子集的pass@1 |
| 20 | 测试覆盖率保持 | 与基分支相比的平均覆盖率变化 |
| 20 | 每迁移仓库成本 | 通过运行的平均$/仓库 |
| 20 | 智能体/确定性工具集成 | OpenRewrite处理 vs 智能体编写的修复比例 |
| 15 | 失败分析报告 | 带示例的分类法完整性 |

## 构建检查清单

- [ ] OpenRewrite/libcst确定性配方
- [ ] Daytona沙箱（每分支）
- [ ] 构建尝试+失败捕获
- [ ] 智能体循环（分类→修复→重试）
- [ ] 预算控制（30分钟/$8/20轮）
- [ ] 测试+覆盖率门控
- [ ] PR打开（成功时）
- [ ] 失败分类法归档
- [ ] 50仓库批量运行+仪表盘
