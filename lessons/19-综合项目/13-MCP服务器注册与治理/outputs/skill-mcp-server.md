# MCP服务器技能

## 目标
构建生产级MCP服务器+注册表+审计层，带OAuth 2.1作用域和OPA门控。

## 评估标准

| 权重 | 标准 | 测量方法 |
|:---:|------|----------|
| 25 | 规范合规 | StreamableHTTP+能力清单通过MCP合规测试 |
| 20 | 安全性 | 作用域执行、OPA覆盖每个工具、密钥卫生 |
| 20 | 可观测性 | 每工具调用审计日志含PII清洗 |
| 20 | 规模 | 100客户端负载测试水平扩展 |
| 15 | 注册表UX | 发现/验证/启用-禁用工作流 |

## 构建检查清单

- [ ] 10个内部工具暴露
- [ ] FastMCP服务器+StreamableHTTP
- [ ] OAuth 2.1作用域检查
- [ ] OPA策略每工具
- [ ] `.well-known/mcp-capabilities`清单
- [ ] 破坏性工具分离服务器
- [ ] 审计日志（JSONL+PII清洗）
- [ ] 注册表UI（发现/验证/启用）
- [ ] 合规套件通过
