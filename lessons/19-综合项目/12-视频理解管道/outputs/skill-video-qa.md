# 视频理解技能

## 目标
给定YouTube URL或上传视频，管道索引场景并回答带时间戳引用的问题。

## 评估标准

| 权重 | 标准 | 测量方法 |
|:---:|------|----------|
| 25 | 时间定位IoU | 保留测试集上的交并比 |
| 20 | QA准确率 | NeXT-GQA和自定义100查询 |
| 20 | 导入吞吐量 | 每美元处理的视频小时数 |
| 20 | UI和引用UX | 时间戳链接、缩略图条、跳转到帧 |
| 15 | 幻觉率 | 计数和动作类准确率单独报告 |

## 构建检查清单

- [ ] 场景分割（TransNetV2/PySceneDetect）
- [ ] VLM描述（每场景关键帧）
- [ ] ASR转录对齐（Whisper-v3-turbo）
- [ ] 多向量Qdrant索引（描述/帧/转录）
- [ ] RRF查询融合
- [ ] TimeLens时间定位
- [ ] VLM答案合成
- [ ] ActivityNet-QA + NeXT-GQA评测
- [ ] 计数和动作类幻觉率单独报告
