# 实时语音助手技能

## 目标
给定领域（客服、排程或信息亭），搭建带有ASR/VAD/LLM/TTS管道的LiveKit智能体，调整到测量标准。

## 评估标准

| 权重 | 标准 | 测量方法 |
|:---:|------|----------|
| 25 | 端到端延迟 | 100次录制通话中首音频输出p50 < 800ms |
| 20 | 轮次质量 | Hamming VAD基准中假切断率 < 3% |
| 20 | 工具正确性 | 对话中工具调用返回正确数据且不卡顿音频 |
| 20 | 丢包可靠性 | 3%丢包下的WER和轮次稳定性 |
| 15 | 评估工具完整性 | 可复现的测量和公开配置 |

## 构建检查清单

- [ ] WebRTC会话（LiveKit room）
- [ ] ASR流式（Deepgram Nova-3或faster-whisper）
- [ ] VAD（Silero v5）
- [ ] 说话人检测（完成评分 > 0.6）
- [ ] LLM流式
- [ ] TTS流式（首字节 < 200ms）
- [ ] 打断处理（Barge-in）
- [ ] 工具侧通道（填充词注入）
- [ ] 评估工具（WER、MOS、延迟）
- [ ] 负载测试（50路并发）
- [ ] 丢包测试（3%丢包）
