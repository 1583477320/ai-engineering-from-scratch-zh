# 智能体循环契约技能

## 目标
实现可复用的智能体循环契约：六状态状态机+十钩子+两拉取点+十一事件+预算包络。

## 评估标准

| 权重 | 标准 | 测量方法 |
|:---:|------|----------|
| 25 | 状态机正确性 | 所有转换符合合法转换表 |
| 20 | 钩子覆盖 | 十个主题全部可触发 |
| 20 | 拉取点恢复 | 从AWAITING_TOOL正确恢复 |
| 20 | 预算执行 | 三个限制正确触发budget.warn |
| 15 | 事件流 | 十一种事件类型正确发射 |

## 构建检查清单

- [ ] 六状态定义+合法转换表
- [ ] 十钩子主题注册和触发
- [ ] 两拉取点（AWAITING_TOOL + on_pause）
- [ ] 十一事件类型发射
- [ ] 三预算限制（轮次、工具调用、墙钟）
- [ ] 确定性重放验证
- [ ] HookAbort异常处理
- [ ] 会话恢复（resume）正确工作
