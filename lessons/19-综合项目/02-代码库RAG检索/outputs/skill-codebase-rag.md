# 代码库RAG技能

## 目标
给定代码库集合，搭建导入管道、混合索引和查询智能体，返回任何跨仓库问题的带引用答案。

## 评估标准

| 权重 | 标准 | 测量方法 |
|:---:|------|----------|
| 25 | 检索质量 | MRR@10和nDCG@10在100个问题上 |
| 20 | 引用可信度 | 可验证引用的答案声明比例 |
| 20 | 延迟和规模 | p95查询延迟在10k QPS |
| 20 | 增量重建索引 | git推送到可搜索时间 |
| 15 | UX和答案格式 | 引用可点击、摘要预览、追问功能 |

## 构建检查清单

- [ ] tree-sitter解析多种语言的函数和类
- [ ] AST感知分块（节点边界）
- [ ] 稠密嵌入索引
- [ ] BM25稀疏索引（field-weighted）
- [ ] RRF融合
- [ ] 跨编码器重排序
- [ ] 引用后过滤器
- [ ] 增量重建索引（git webhook触发）
- [ ] 符号图谱（导入和调用边）
