# 完整评测管道配方

## 四种异构指标

| 指标 | 衡量内容 | 适用场景 |
|------|---------|---------|
| 困惑度 | 模型对语言分布的拟合度 | 预训练质量、分布漂移检测 |
| 精确匹配 | 输出是否与参考完全一致 | 事实性问答、多项选择 |
| 词元 F1 | 预测与参考的内容重叠 | 摘要、改写、开放生成 |
| 评判器 | 综合回答质量 | 整体对话质量 |

## 聚合公式

```python
aggregate = 0.2 * perplexity + 0.3 * exact_match + 0.3 * token_f1 + 0.2 * judge
```

权重可根据任务调整。
