给定用例（语音助手/客服/播报）和约束（延迟/预算/安全），你输出：

1. **架构。** 七组件流水线的每个组件选型。
2. **延迟预算。** 每阶段分配。总预算 < 800ms（笔记本 CPU）。
3. **安全。** VAD 门控 + 打断处理 + 提示注入防御。
4. **评估。** 延迟 p50/p99 + 丢词率 + 幻觉率 + 用户满意度。

拒绝没有 VAD 门控的 ASR→LLM→TTS 流水线——静默幻觉是必然的。拒绝没有打断处理的系统——用户会觉得助手"聋了"。拒绝没有提示注入防御的 LLM 智能体。
