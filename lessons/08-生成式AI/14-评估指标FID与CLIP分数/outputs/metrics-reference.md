# 生成模型评估指标参考表

## FID 值参考

| FID 范围 | 质量评估 | 代表模型 |
|---------|---------|---------|
| 0 | 完美匹配（与真实分布相同） | — |
| 1-5 | 优秀（几乎无法区分） | Sora, DALL-E 3 |
| 5-10 | 良好（高质量，轻微差异） | Stable Diffusion, Midjourney |
| 10-30 | 可接受（训练良好） | DDPM, StyleGAN 2 |
| 30-50 | 一般（有明显差异） | VAE |
| > 50 | 差（分布显著不同） | 欠拟合模型 |

## CLIP Score 参考

| CLIP Score | 语义对齐 | 评估 |
|-----------|---------|------|
| < 20 | 差 | 图像与提示词不匹配 |
| 20-25 | 弱 | 大致方向正确，细节偏差 |
| 25-30 | 中 | 主体正确 |
| 30-33 | 良好 | 主体+大部分细节正确 |
| 33-35 | 优秀 | 高度符合提示词描述 |
| > 35 | 极好 | 几乎精确匹配 |

## LPIPS 参考

| LPIPS | 感知相似度 |
|-------|-----------|
| 0 | 完全相同 |
| 0-0.1 | 几乎相同 |
| 0.1-0.3 | 略有差异 |
| 0.3-0.5 | 明显差异 |
| > 0.5 | 完全不同的图像 |

## 标准差（不同样本量）

| 样本量 | FID 标准差 |
|-------|-----------|
| 500 | ±3.0 |
| 1000 | ±1.5 |
| 5000 | ±0.5 |
| 10000 | ±0.3 |
| 50000 | ±0.1 |

## 常见陷阱

1. FID 对分辨率敏感——不同分辨率下的 FID 不可比
2. CLIP Score 对提示词措辞敏感——相同含义不同措辞得分不同
3. 两个指标都在 ImageNet 域有效——迁移到其他域需要验证
4. "FID 降低 1 分不等价于 CLIP Score 提高 1 分"
