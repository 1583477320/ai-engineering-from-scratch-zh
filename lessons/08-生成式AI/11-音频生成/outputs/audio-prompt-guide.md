# 音频生成提示词指南

## 用途

面向中文用户的 TTS（语音合成）、音乐生成、音效生成提示词编写指南。

## 一、TTS（语音合成）

### 文本预处理

TTS 模型对输入文本敏感，必须先清洗：

```python
def clean_tts_text(text):
    text = text.replace("  ", " ")  # 合并空格
    # 替换特殊符号为可读描述
    text = text.replace("https://...", "网址")
    text = text.replace("...", "省略号")
    # 英文缩写拼读
    text = text.replace("AI", "人工智能")
    text = text.replace("API", "接口")
    text = text.replace("CPU", "C P U")
    return text
```

### 情感/风格控制

| 描述 | 效果 |
|------|------|
| "用温和的语气说" | 降低语速，减少起伏 |
| "用兴奋的语气说" | 提高语速和音高范围 |
| "用低沉的语气说" | 降低基础音高 |
| "用紧急的语气说" | 加快语速，提高能量 |

## 二、音乐生成

### MusicGen 提示词结构

```
[风格] [乐器] [节奏] [情绪] [结构]
```

示例：
```
80年代合成波，合成器主导，BPM 120，充满希望和怀旧感，前奏-主歌-副歌-结尾
```

### 常见音乐风格标签

| 中文 | 英文 |
|------|------|
| 古典 | classical, orchestral |
| 爵士 | jazz, smooth jazz |
| 摇滚 | rock, hard rock, punk |
| 流行 | pop, synth-pop, K-pop |
| 电子 | electronic, house, techno |
| 中国风 | Chinese traditional, guzheng, erhu |
| 放松 | ambient, lo-fi, chill |
| 电影配乐 | cinematic, orchestral, epic |

### 节奏控制

| BPM | 效果 |
|-----|------|
| 60-80 | 慢速（抒情、氛围） |
| 80-120 | 中速（流行、摇滚） |
| 120-150 | 中快速（舞曲、EDM） |
| 150-180 | 快速（金属、硬核） |

## 三、音效生成

### AudioLDM 提示词结构

```
[环境] + [主要声音] + [次要声音] + [质量]
```

示例：
```
强烈的雷雨声，远处有犬吠，自然环境音，高保真
```

### 常见音效分类

| 类别 | 示例提示词 |
|------|-----------|
| 自然 | 雨声、鸟鸣、海浪、风声 |
| 城市 | 车流、人群、警报、地铁 |
| 动物 | 狗叫、猫叫、鸟鸣 |
| 机械 | 引擎声、键盘声、铃声 |
| 环境 | 咖啡馆、办公室、森林 |

## 四、常见错误

- ❌ TTS 输入含表情符号（😊）→ 模型忽略或输出异常
- ✅ 清洗文本：`"这篇很重要😊"` → `"这篇很重要"`
- ❌ 音乐提示词太简单（"一首歌"）→ 输出随机
- ✅ 指定风格+乐器+节奏（"R&B，钢琴和贝斯，90BPM，浪漫"）
- ❌ 音效提示词太长（不是写作文）
- ✅ 简洁描述环境（"雨声，窗户上的水滴声"）
