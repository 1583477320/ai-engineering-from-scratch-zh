# LoRA 训练参数选择和调优指南

## 用途

指导如何选择 LoRA 的训练参数（rank、alpha、学习率、轮次），以获得最佳微调效果。

## 参数选择决策树

```
你的目标是什么？
├── 单一风格/材质（如水彩、素描）
│   └── rank=4~8, alpha=4~8, epochs=10~20
├── 多角色/多场景
│   └── rank=16~32, alpha=16~32, epochs=20~40
├── 特定人物/角色
│   └── rank=8~16, alpha=8~16, epochs=15~30
└── 复杂艺术风格（如毕加索立体主义）
    └── rank=32~64, alpha=32~64, epochs=30~50
```

## 关键参数说明

### Rank（秩）

| Rank | 参数量（SD 1.5） | 文件大小 | 适用场景 |
|------|-----------------|---------|---------|
| 4 | ~256K | ~1MB | 简单纹理、滤镜 |
| 8 | ~512K | ~2MB | 单一风格 |
| 16 | ~1MB | ~4MB | 通用选择（推荐） |
| 32 | ~2MB | ~8MB | 复杂风格 |
| 64 | ~4MB | ~16MB | 多角色/多场景 |
| 128 | ~8MB | ~32MB | 接近全量微调 |
| 256 | ~16MB | ~64MB | 不推荐（接近全量） |

### Alpha（缩放因子）

- **alpha = rank**（默认）：缩放因子 = 1.0，最常用
- **alpha = 2 * rank**：增强 LoRA 效果，适合简单风格
- **alpha = rank / 2**：减弱 LoRA 效果，适合复杂风格或多 LoRA 叠加

### 学习率

| 学习率 | 效果 | 风险 |
|-------|------|------|
| 1e-3 | 快速收敛 | 可能过拟合 |
| 1e-4 | 推荐默认 | 平衡 |
| 1e-5 | 保守训练 | 可能欠拟合 |
| 5e-6 | 极低 | 需要更多轮次 |

### 轮次（Epochs）

| 数据集大小 | 推荐轮次 | 总步数（batch=1） |
|-----------|---------|-----------------|
| 10 张图片 | 15-25 | 150-250 |
| 50 张图片 | 10-20 | 500-1000 |
| 200 张图片 | 15-30 | 3000-6000 |
| 500+ 张图片 | 20-50 | 10000-25000 |

## 过拟合检测

训练过程中观察以下信号：

1. **生成的图像过于贴近训练集**：所有输出都看起来像训练图片
2. **loss 持续下降但生成质量不再提升**：模型在记忆而非学习
3. **失去基础模型的通用能力**：不用 LoRA 时生成的图像质量下降

## 调优建议

- 如果图像模糊 → 降低 learning_rate 或增加 rank
- 如果图像过于刻板 → 降低 alpha 或降低 conditioning_scale
- 如果训练时间过长 → 降低 rank 或使用更小的 batch
- 如果效果不稳定 → 增加 dropout（0.1-0.3）
