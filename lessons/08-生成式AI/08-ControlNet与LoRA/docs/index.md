# ControlNet 与 LoRA

> ControlNet 让扩散模型遵循结构化控制（边缘图、深度图、姿态）。LoRA 让微调只更新 1% 的参数。两者结合 = 用极少计算获得精确控制的生成。

**类型：** 概念课 | **语言：** Python | **前置知识：** 阶段 08 · 07（潜在扩散）| **时间：** ~60 分钟

---

## 🎯 学习目标

- [ ] 理解 ControlNet 的零卷积初始化——为什么它能"附加"到预训练模型而不破坏原始权重
- [ ] 解释 LoRA 的低秩分解——如何用 1-2% 的参数实现微调
- [ ] 说明 ControlNet + LoRA 的组合——边缘检测+艺术风格

## 1. 问题

Stable Diffusion 的文本条件化很好——但它不知道"在哪里画"。ControlNet 通过添加空间控制信号（边缘图、深度图、姿态骨架）解决了这个问题。LoRA 让你在不全量微调的情况下注入新能力——比如特定艺术家的画风。

**ControlNet + LoRA = 最小成本的定制生成。**

---

## 2. 概念

### 2.1 ControlNet

```
控制信号（如边缘图）→ [可训练副本 + 零卷积] → 交叉注意力注入 U-Net
```

- **零卷积初始化：** ControlNet 开始时输出为零——不破坏预训练模型
- **克隆 U-Net 的一半：** 可训练的"编码器"副本学习控制信号
- **支持的控制信号：** Canny 边缘、深度图、姿态骨架、涂鸦、分割图

### 2.2 LoRA

```
原始层：W ∈ (d, d)    → 固定，不训练
LoRA 层：A ∈ (d, r) × B ∈ (r, d)  → 可训练，r << d
更新：W + α·A·B    → 只训练 A 和 B
```

- **秩 r=16：** 每层可训练参数 = 2×d×r，通常 < 1%
- **多适配器：** 可以为不同风格训练多个 LoRA，在推理时切换

---

## 🔑 关键术语

| 术语 | 含义 |
|---|---|
| 零卷积 | ControlNet 开始时权重为零——不破坏预训练模型 |
| LoRA | 低秩自适应——用 A×B 低秩分解替代全参数更新 |
| ControlNet | 向扩散模型注入空间控制信号（边缘/深度/姿态） |

---

## 📚 小结

ControlNet 通过零卷积初始化和 U-Net 克隆副本，向预训练扩散模型注入空间控制。LoRA 用低秩分解将微调参数降到 1-2%。两者组合 = 用极小成本获得精确控制的生成——边缘图控制位置 + LoRA 控制风格。
