# 3D 生成提示词指南

## 用途

指导如何为文本到 3D 生成模型（Meshy、Shap-E、TripoSR）编写有效的提示词。

## 核心原则

1. **描述形状先于材质**——模型需要知道"是什么"再知道"像什么"
2. **指定视角方向**——3D 需要前后左右一致
3. **避免抽象描述**——"奇幻风格"太模糊，换成具体描述

## 提示词模板

### 基础模板

```
[物体类型] + [形状特征] + [材质] + [颜色] + [细节] + [风格]
```

示例：
```
一把现代风格办公椅，网面靠背，黑色金属框架，五爪轮底座，人体工学设计
```

### Meshy（商业级）提示词

| 意图 | 提示词 |
|------|--------|
| 家具 | "一张中式红木茶几，圆形桌面，雕花桌腿，抛光表面" |
| 建筑 | "一座现代别墅，大面积玻璃幕墙，平屋顶，白色外墙" |
| 人物 | "一个卡通风格的人物，大眼微笑，穿蓝色T恤" |
| 道具 | "一把英雄风格的长剑，银色剑身，金色护手，镶嵌红宝石" |

### 多视角提示词（用于生成多视角图像→TripoSR）

```
正面：一只蓝色小鸟的正面照片，背景白色
侧面（左）：蓝色小鸟的左侧面照片
侧面（右）：蓝色小鸟的右侧面照片
背面：蓝色小鸟的背面照片
```

## 材质描述词汇

| 中文 | 英文 |
|------|------|
| 木质纹理 | wooden texture, wood grain |
| 金属光泽 | metallic, reflective |
| 半透明 | translucent, semi-transparent |
| 粗糙表面 | rough, matte |
| 光滑表面 | smooth, glossy |
| 磨砂 | frosted, matte finish |
| 皮革质感 | leather, textured |

## 常见错误

- ❌ "一个很酷的东西" → 模型不认识"很酷"
- ✅ "一把未来风格的步枪，流线型枪身，蓝色发光细节" → 具体描述
- ❌ 提示词太短（1-3个词）→ 输出随机
- ✅ 2-3行描述 → 输出可控
