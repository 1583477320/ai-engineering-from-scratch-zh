# AI 代码调试指南

## 调试顺序

1. **训练前**：用 `debug_print` 检查样本批次的形状和数据类型
2. **前 10 步**：检查损失、输出和梯度的 NaN 和值范围
3. **训练中**：用 TensorBoard 记录损失、学习率、梯度范数
4. **出错时**：在失败点放置 `breakpoint()` 交互检查张量
5. **性能**：分析数据加载 vs 前向 vs 反向传播的时间比例

## 常见 Bug

| Bug | 现象 | 检测 |
|:----|:-----|:-----|
| 形状不匹配 | 运行时维度错误 | forward hook |
| NaN 损失 | 训练发散 | `torch.isnan(loss)` |
| 数据泄漏 | 测试 99% 准确率 | 检查 ID 重叠 |
| 设备不匹配 | 训练极慢 | 检查 `.device` |
