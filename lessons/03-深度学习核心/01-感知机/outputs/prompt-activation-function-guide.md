# 提示词：激活函数选择指南

## 用途

将此提示词发送给 Claude / GPT，获取针对具体场景的激活函数选择建议。

---

## 提示词

```
你是一个深度学习专家。我正在设计一个神经网络，请帮我选择激活函数。

请按以下结构回答：

## 1. 任务分析
根据我描述的任务类型，分析输出层需要什么激活函数。

## 2. 隐藏层推荐
列出 2-3 种适合该任务的隐藏层激活函数，比较它们的优劣。

## 3. 具体建议
给出最终推荐方案，包括：
- 输出层用什么
- 隐藏层用什么
- 权重初始化用什么（Xavier / He / 其他）
- 学习率的合理起始范围

## 4. 代码示例
用 PyTorch 写一个 nn.Sequential 示例，体现你的推荐。

## 5. 常见陷阱
列出该场景下最容易踩的 2-3 个坑。

---

我的任务描述：[在此处填写你的任务，例如"中文文本情感分类，二分类"]
```

---

## 使用示例

**输入：** "中文文本情感分类，二分类"

**预期输出结构：**

```
## 1. 任务分析
二分类任务，输出层需要一个输出单个概率值的激活函数...

## 2. 隐藏层推荐
- ReLU：计算简单，收敛快，但可能出现"神经元死亡"
- GELU：平滑版 ReLU，BERT/GPT 标配
- Tanh：输出以 0 为中心，适合 RNN

## 3. 具体建议
输出层用 Sigmoid，隐藏层用 GELU...
```

---

## 适用场景

- 设计新网络架构时
- 调试模型不收敛时
- 比较不同激活函数的效果时
