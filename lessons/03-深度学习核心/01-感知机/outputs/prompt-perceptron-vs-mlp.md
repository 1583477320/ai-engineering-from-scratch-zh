# 提示词：感知机 vs 多层网络选择指南

## 用途

将此提示词发送给 Claude / GPT，分析你的问题是否需要多层网络。

---

## 提示词

```
你是一个神经网络架构顾问。我正在分析一个问题，判断用单层感知机还是多层感知机（MLP）。

请按以下结构回答：

## 1. 线性可分性判断
根据我描述的问题，判断它是否线性可分。给出推理过程。

## 2. 单层感知机方案
如果线性可分，给出单层感知机的：
- 输入维度
- 权重数量
- 训练轮次估计
- 预期准确率

## 3. 多层感知机方案
如果非线性可分，给出 MLP 的：
- 建议的层数和每层神经元数
- 激活函数选择
- 权重总数估计
- 训练注意事项

## 4. 决策边界可视化描述
用文字描述单层和多层方案的决策边界形状差异。

## 5. 最终建议
一句话总结：用单层还是多层？为什么？

---

我的问题描述：[在此处填写你的问题，例如"判断一个二维坐标点是否在单位圆内"]
```

---

## 使用示例

**输入：** "判断一个二维坐标点是否在单位圆内"

**预期输出结构：**

```
## 1. 线性可分性判断
单位圆内的判断是非线性可分的。圆内（x² + y² < 1）和圆外
无法用单条直线分隔。需要至少一个隐藏层。

## 2. 单层感知机方案
不适用。单层感知机只能画直线，无法拟合圆形边界。

## 3. 多层感知机方案
建议 2-4-1 架构：
- 输入层：2 个神经元（x, y）
- 隐藏层：4 个 ReLU 神经元
- 输出层：1 个 Sigmoid 神经元
权重总数：2×4 + 4 + 4×1 + 1 = 17 个

## 4. 决策边界可视化描述
单层：一条直线，最多达到 75% 准确率
多层：可以逼近圆形边界，理论上 100% 准确

## 5. 最终建议
必须用多层感知机。圆形边界是非线性的，
单层感知机的直线无法拟合。
```

## 适用场景

- 评估新问题的网络复杂度需求
- 判断是否可以用简单模型
- 向团队解释架构选择的原因
