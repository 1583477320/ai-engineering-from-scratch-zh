# Q 学习与 SARSA

> TD(0) 是 RL 的"核心算子"——从一次经验中更新价值估计，不需要等到轨迹结束。Q 学习是"离策略"的 TD——SARSA 是"在策略"的 TD。

**类型：** 实现课 | **语言：** Python | **前置知识：** 阶段 09 · 01（MDP）、02（动态规划）、03（蒙特卡洛）| **时间：** ~75 分钟

## 1. 问题

蒙特卡洛需要完整轨迹——不能在线学习。TD(0) 解决这个问题：**用当前的价值估计来替代未来回报**——单步更新。

$$V(s) \leftarrow V(s) + \alpha [r + \gamma V(s') - V(s)]$$

Q 学习将这个思想扩展到 Q 值——并用**贪心目标**而非实际行为来更新（离策略）。

$$Q(s,a) \leftarrow Q(s,a) + \alpha [r + \gamma \max_{a'} Q(s',a') - Q(s,a)]$$

SARSA（在策略）用**实际选择的动作**更新：

$$Q(s,a) \leftarrow Q(s,a) + \alpha [r + \gamma Q(s',a') - Q(s,a)]$$

---

## 🔑 关键术语

| 术语 | 含义 |
|---|---|
| TD(0) | 单步时序差分——用当前估计替代未来回报 |
| Q 学习 | 离策略 TD——用贪心目标更新 Q 值 |
| SARSA | 在策略 TD——用实际行为更新 Q 值 |
| 离策略 | 评估策略≠行为策略——可以学习比当前行为更优的策略 |

---

## 📚 小结

TD(0) = 蒙特卡洛的在线版本——单步自举替代完整轨迹。Q 学习 = 离策略 TD——用贪心目标更新 Q 值，收敛到最优策略。SARSA = 在策略 TD——更保守但更安全。ε-贪心探索是连接两者的桥梁。
