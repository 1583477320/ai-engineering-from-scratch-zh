## 1. 问题

Q 学习用 Q 表存储每个 (状态,动作) 的价值。但在 Atari 游戏中，状态是 84×84×4 的像素帧——维度爆炸。DQN 的答案：**用神经网络替代 Q 表**——输入状态，输出每个动作的价值。

### DQN 的两个关键技巧

1. **经验回放：** 将 (s, a, r, s') 存入缓冲区，随机采样训练——打破时间相关性
2. **目标网络：** 用固定的目标网络计算目标 Q 值——每 C 步才同步——防止不稳定振荡

---

## 🔑 关键术语

| 术语 | 含义 |
|---|---|
| 经验回放 | 存储轨迹并在训练时随机采样——打破时间相关性 |
| 目标网络 | 固定的 Q 值计算网络——每 C 步才与主网络同步 |
| ε-贪心 | 以概率 ε 随机动作，否则选 Q 值最大的动作 |

## 📚 小结

DQN 用神经网络替代 Q 表——从原始像素学习动作价值。经验回放打破时间相关性；目标网络稳定训练。2013 年在 Atari 上超越人类——这是深度 RL 的起点。
