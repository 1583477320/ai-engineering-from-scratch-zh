# 位置编码对比指南

> 比较正弦编码、可学习编码和 RoPE 的优缺点。

## 三种位置编码对比

| 特性 | 正弦编码 | 可学习编码 | RoPE |
|------|----------|------------|------|
| 参数数量 | 0 | L × d_model | 0 |
| 外推能力 | ✓ 可以外推 | ✗ 固定长度 | ✓ 可以外推 |
| 相对位置 | 间接（线性变换） | 间接 | ✓ 直接（旋转矩阵） |
| 计算开销 | 低 | 低 | 中 |
| 使用模型 | 原始 Transformer | BERT, GPT-2 | GPT-4, Llama 3 |

## 选择建议

1. **学习/实验**：正弦编码——简单易理解
2. **固定长度任务**：可学习编码——可能学到更好的位置表示
3. **长序列/生成任务**：RoPE——外推能力强，相对位置信息更直接

## 关键观察

- 正弦编码的相邻位置距离很小，远距离位置距离较大
- 可学习编码的位置距离没有明显规律
- RoPE 通过旋转矩阵让注意力分数直接反映相对位置
