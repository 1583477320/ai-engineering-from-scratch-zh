# CNN vs ViT 对比指南

> 两种主流视觉架构的详细对比。

## 架构对比

| 维度 | CNN | ViT |
|------|-----|-----|
| 核心操作 | 卷积（局部连接） | 自注意力（全局连接） |
| 归纳偏置 | 局部性、平移不变性 | 无先验 |
| 感受野 | 随层数增大 | 全局（第一层开始） |
| 数据需求 | 较少 | 较多 |
| 参数量（相似性能） | 较少 | 较多 |
| 推理速度 | 快 | 中 |
| 迁移学习 | 好 | 需要更多数据 |
| 长距离依赖 | 弱 | 强 |

## 参数量对比

| 模型 | 参数量 | 架构 |
|------|--------|------|
| ResNet-50 | 25M | CNN |
| ResNet-152 | 60M | CNN |
| ViT-Base | 86M | Transformer |
| ViT-Large | 307M | Transformer |
| ViT-Huge | 632M | Transformer |

## 选择建议

| 场景 | 推荐架构 |
|------|----------|
| 小数据集（<10K 图片） | CNN（更强的归纳偏置） |
| 大数据集（>100K 图片） | ViT（更强的表达能力） |
| 边缘设备 | CNN（更小、更快） |
| 语义分割 / 目标检测 | CNN 或 CNN+Transformer 混合 |
| 自监督学习 | ViT（DINOv2） |
| 多模态（文本+图像） | ViT（与 NLP 模型一致） |

## 混合架构

| 模型 | 设计 |
|------|------|
| ConvNeXt | 用 CNN 实现 ViT 风格设计 |
| MaxViT | CNN + Transformer 混合 |
| Swin Transformer | 层级式 Transformer，窗口注意力 |
