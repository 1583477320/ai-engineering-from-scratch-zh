## 1. 问题

你读到 PCA 看到"找到协方差矩阵的特征向量"。读到模型稳定性看到"检查所有特征值的模是否小于 1"。读到数据增强看到"应用随机旋转"。在理解矩阵对空间的几何作用之前，这些都不成立。

矩阵不只是数字网格。它们是空间机器。旋转矩阵旋转点，缩放矩阵拉伸点，剪切矩阵倾斜点。神经网络对数据的每个变换都是这些操作之一或它们的组合。

## 2. 核心概念

### 2.1 四种基本变换

| 变换 | 矩阵 | 几何效果 |
|:-----|:-----|:---------|
| 旋转 | `[[cos θ, -sin θ], [sin θ, cos θ]]` | 沿圆弧移动点，保持距离和角度 |
| 缩放 | `[[sx, 0], [0, sy]]` | 沿各轴独立拉伸/压缩 |
| 剪切 | `[[1, k], [0, 1]]` | 一个轴倾斜，另一个不动 |
| 反射 | `[[-1, 0], [0, 1]]` | 跨轴镜像点 |

### 2.2 组合变换

应用变换 A 然后 B 等同于矩阵乘法：`result = B @ A @ point`。**顺序很重要**——先旋转再缩放 vs 先缩放再旋转产生不同结果。

### 2.3 特征值和特征向量

大多数向量在矩阵作用下改变方向。特征向量是特殊的：矩阵只缩放它们，不旋转。缩放因子就是特征值。

```
A @ v = λ × v

v = 特征向量（存活的方向）
λ = 特征值（拉伸多少）

例子: A = [[2,1],[1,2]]
特征向量 [1,1] 特征值 3:  A@[1,1] = [3,3] = 3×[1,1]
特征向量 [1,-1] 特征值 1:  A@[1,-1] = [1,-1] = 1×[1,-1]
```

矩阵沿 [1,1] 拉伸 3 倍，[1,-1] 不变。

### 2.4 特征分解

```
A = V @ D @ V⁻¹
V = 特征向量矩阵
D = 特征值对角矩阵
含义：旋转到特征向量坐标，沿每轴缩放，再旋转回来
```

### 2.5 特征值为什么重要

- **PCA**：协方差矩阵的特征向量是主成分。特征值告诉每个成分捕获多少方差
- **RNN 稳定性**：特征值模 > 1 导致爆炸，< 1 导致消失——梯度消失/爆炸问题一句话总结
- **谱方法**：图神经网络用邻接矩阵的特征值，谱聚类用拉普拉斯矩阵

## 3. 从零实现

```python
"""矩阵变换——旋转、缩放、特征值。"""
import math

def rotation_2d(theta):
    c, s = math.cos(theta), math.sin(theta)
    return [[c, -s], [s, c]]

def scaling_2d(sx, sy):
    return [[sx, 0], [0, sy]]

def mat_vec_mul(M, v):
    return [sum(M[i][j] * v[j] for j in range(len(v))) for i in range(len(M))]

def mat_mul(A, B):
    return [[sum(A[i][k]*B[k][j] for k in range(len(A[0]))) for j in range(len(B[0]))]
            for i in range(len(A))]

def eigenvalues_2x2(matrix):
    a, b = matrix[0]; c, d = matrix[1]
    trace, det = a + d, a * d - b * c
    disc = trace**2 - 4 * det
    if disc < 0:
        return (complex(trace/2, (-disc)**0.5/2), complex(trace/2, -(-disc)**0.5/2))
    sd = disc**0.5
    return ((trace + sd) / 2, (trace - sd) / 2)

# 演示
point = [1.0, 0.0]
R = rotation_2d(math.pi / 4)
print(f"旋转 45°: {mat_vec_mul(R, point)}")

A = [[2, 1], [1, 2]]
vals = eigenvalues_2x2(A)
print(f"特征值: {vals[0]:.4f}, {vals[1]:.4f}")

R90 = rotation_2d(math.pi/2)
S = scaling_2d(2, 0.5)
print(f"\n先旋转后缩放: {mat_vec_mul(S, mat_vec_mul(R90, point))}")
print(f"先缩放后旋转: {mat_vec_mul(R90, mat_vec_mul(S, point))}")
```

## 7. 常见错误

### 错误 1：组合变换顺序搞反

**现象：** `A @ B` 与 `B @ A` 结果不同但未意识到。

**原因：** 矩阵乘法不可交换——先旋转再缩放 vs 先缩放再旋转。

**修复：** 始终检查变换的应用顺序——`result = B @ A @ point` 意味着先 A 后 B。

### 错误 2：特征值为复数

**现象：** `disc < 0` 时特征值返回复数。

**原因：** 实矩阵的特征值可以是复数——对应旋转而非缩放。

**修复：** 复数特征值的模为旋转角度，实部为缩放因子。

## 8. 面试考点

### Q1：为什么特征值决定 RNN 的梯度爆炸/消失？（难度：⭐⭐⭐）

**参考答案：** RNN 的隐状态更新为 $h_t = f(W h_{t-1} + b)$。经过 T 步，梯度包含 $W^T$ 的 T 次幂。$W$ 的特征值 λ 的 T 次幂是 $λ^T$。如果 |λ| > 1，$λ^T$ 指数增长→爆炸。如果 |λ| < 1，$λ^T$ 指数衰减→消失。只有 |λ| ≈ 1 时梯度才能稳定传播。这就是为什么初始化权重和选择架构如此重要。

## 🔑 关键术语

| 术语 | 含义 |
|:-----|:-----|
| 旋转矩阵 | 正交矩阵，行列式 = 1，保持距离 |
| 特征向量 | 矩阵只缩放不旋转的方向 |
| 特征值 | 矩阵对其特征向量的缩放因子 |
| 特征分解 | A = V @ D @ V⁻¹，将矩阵分解为旋转+缩放 |
| 行列式 | 变换缩放面积/体积的因子。零 = 不可逆 |
