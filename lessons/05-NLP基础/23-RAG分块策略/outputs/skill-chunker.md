给定文档类型（结构化/非结构化、语言、平均长度），你输出：

1. 分块策略。递归（默认——按 `\n\n`→`\n`→`。`→`；`→` ` 优先级）。文档结构（Markdown 标题/HTML 标签）。句子窗口（精准匹配 + 上下文）。语义（仅当递归不达预期）。
2. 参数。块大小：256-1024 token（嵌入模型窗口的 70%）。重叠：10-20% 或 1-3 个完整句子。2026 基准：递归固定大小 > 语义分块（69% vs 54%）。
3. 元数据。每块携带：文档名、页码/章节标题、块在文档中的位置。
4. 验证。在 20 条目标领域真实查询上测 Recall@5。检查是否有任何答案的关键信息恰好跨块边界丢失。

中文：按 `。`→`；`→`，` 优先级分割。重叠以完整句子为单位（2-3 句），不以 token 数——中文 token/字比例在不同文体中差异大。
