给定场景（领域、语言、对准确性的要求），你输出：

1. 方案。英文生产：spaCy + `experimental_coref` / FastCoref。中文：目前无生产可用的开源方案覆盖零指代——规则+距离约束（封闭域）或 LLM prompt（开放域）。
2. 中文零指代策略。代词消解：规则（"它"→最近的非人实体，"他"→最近的男性人名）。省略主语：LLM prompt "以下对话中省略的主语是谁？"。
3. 验证。抽取 30 个含代词/指代的段落，人工标注指代链，对比模型输出。
4. 一个陷阱。指代消解在跨句距离 > 3 句时准确率急剧下降。长文档中远距离的"该公司"可能被误链到另一个公司。

拒绝假设中英文指代消解难度相同（中文零指代是英文工具无法处理的额外挑战）。
