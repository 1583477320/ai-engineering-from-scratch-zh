# 指代消解

> "她打电话给他。他没接。医生在吃午饭。"三个指称、两个人、没有一个是名字。指代消解弄清楚谁是谁。

**类型：** 概念课 | **语言：** Python | **前置知识：** 阶段 05 · 06、05 · 07 | **预计时间：** ~60 分钟 | **所处阶段：** Tier 1

## 1. 问题

从一篇 300 字的文章中提取所有关于苹果公司的提及。文章写"Apple"时很好找。写"这家公司"、"他们"、"库比蒂诺的科技巨头"、"乔布斯的公司"时——NER 管道漏掉了 60-80% 的提及。

指代消解将指向同一现实实体的每个表达链接到一个簇中。它是表层 NLP（NER、句法分析）和下游语义（信息提取、QA、摘要、知识图谱）之间的胶水：

- **摘要：** "CEO 宣布…" vs "Tim Cook 宣布…"——摘要应该写出人名
- **问答：** "她给谁打了电话？"——需要先消解"她"
- **信息提取：** 知识图谱中"PER1 创立苹果"和"乔布斯 创立 苹果"是重复条目——消解后将它们合并

---

## 2. 指代类型

| 类型 | 示例 |
|---|---|
| 代词 | "苹果发布了新 iPhone。**它**比上一代快两倍。" |
| 定指名词短语 | "特斯拉股价上涨。**这家电动车制造商**表示…" |
| 专有名词重复 | "Tim Cook 参加了会议。Cook 表示…" |
| 零指代 (中文特有) | "张三走进房间，Ø 坐下来，Ø 打开了电脑。"——后两句的主语被省略 |

### 中文零指代——英文没有的挑战

中文是**代词脱落（Pro-drop）**语言——当主语从上下文可以推断时，代词可以被省略。英文中你会写"John went into the room. **He** sat down."中文中"张三走进房间。**Ø** 坐下来。"第二个句子的主语完全没有显式表达。中文指代消解必须额外处理这些"隐形"的指代——这是英文工具无法直接迁移的根本原因。

### 中文指代消解

中文指代消解的工具比英文少一个数量级——目前没有达到生产可用的开源方案能覆盖零指代。**2026 年的务实做法：** 对封闭域（如客服对话中的"它"→最近一次提到的产品），用规则 + 距离约束消解代词。对开放域——使用 LLM prompt："以下对话中的'他'指的是谁？输出实体名。"
